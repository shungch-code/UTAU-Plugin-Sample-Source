UTAU Plugin Sample Source

Welcome to the UTAU-Plugin-Sample-Source project. This repository provides the original UTAU sample plugin by Ameya/Ayame (2008), modified for research and educational use.

Overview

  • Read and modify UST note parameters
  • Display all parameters in a dialog Edit Box
  • Support pitch bend (PBW, PBS, PBY, PBM)
  • Envelope shaping and flag-based timbre control

How to Use

  1. Download or compile the plugin (Visual C++ 6.0).
  2. Copy utauplugin.exe into the UTAU plugins folder.
  3. Open UTAU, select notes, and run the plugin.
  4. Edit parameters directly in the dialog box.
